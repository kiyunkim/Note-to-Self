--|--

## Types of Patterns

### Creational

Patterns that deal with the creation of new instances of an object

1. Constructor pattern
2. Module pattern
3. Factory
4. Singleton

### Structural

Deal with the makeup of the object
1. Decorator
2. Façade
3. Flyweight 

### Behavioral

How objects operate and interact with each other
1. Command
2. Mediator
3. Observer
